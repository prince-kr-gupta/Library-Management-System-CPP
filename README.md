📚 Library Management System in C++

This is a simple console-based Library Management System developed in C++ using Object-Oriented Programming (OOP) concepts and file handling.

I built this project as part of my learning journey in C++ to strengthen my understanding of classes, inheritance, file handling, and real-world problem solving. The project simulates a basic library where an admin can manage books and students can borrow or return them.

✨ Features
Admin Panel
Secure Admin Login
Add New Books
Display All Books
Update Book Details
Remove Books from Library
Student Panel
Student Login
View Available Books
Borrow Books
Return Borrowed Books
File Handling
Book records are stored in BookData.txt
Borrowed book records are stored in BorrowData.txt
Data remains saved even after closing the program
🛠 Technologies Used
C++
Object-Oriented Programming (OOP)
File Handling (fstream)
Console-Based User Interface

🚀 How to Run
Clone the repository.
Open the project in your preferred C++ IDE.
Compile and run main.cpp.
Follow the menu options displayed on the screen.
